# How to Update this Project on GitHub

1.  **Check Status**: open a terminal and run this command to see which files have changed.
    ```bash
    git status
    ```

2.  **Add Changes**: run this command to stage all your changes for commit.
    ```bash
    git add .
    ```

3.  **Commit Changes**: run this command to save your staged changes with a descriptive message.
    ```bash
    git commit -m "Description of your changes"
    ```

4.  **Push to GitHub**: run this command to upload your commits to the remote repository.
    ```bash
    git push origin main
    ```

If you need to get the latest changes from GitHub before pushing, run:
```bash
git pull origin main
```
